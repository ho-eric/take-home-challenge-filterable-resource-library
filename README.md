# Take Home Challenge: Filterable Resource Library

To run the project locally, you will need to use a local development server due to many modern brower's security settings blocking the javascript and json file due to CORS settings.
I used VS Code along with the Live Preview plugin

Additionally, I have also created a site that is deployed from this repo here:  https://ho-eric.github.io/take-home-challenge-filterable-resource-library/

One assumption I made was that as the default state, with no filters or search term entered, the site would show ALL of the resources.

If I had more time, I would add a button to reset all filters. I would also add a couple more subtle animations to flesh out the look and feel of the site. 

I spent approximately 4 hours working on the assignment
